# ☕ SSAFY COFFEE

> SSAFY 광주캠퍼스 메가커피 단체 주문 관리 서비스

## 소개

매일 점심 전 팀원들의 메가커피 주문을 한 곳에서 모아 관리하는 웹 서비스입니다.<br/>
키오스크 스타일의 메뉴 선택, 실시간 주문 목록 조회, 오늘의 픽업 당첨자 추첨까지 지원합니다.

## 주요 기능

### 주문하기
- 이름, 반 입력 후 메뉴를 키오스크 버튼으로 선택 (재방문 시 자동 기입)
- 카테고리 탭으로 메뉴 분류, 선택 시 이미지 표시
- 페이지네이션으로 다수의 메뉴 탐색
- ICE / HOT 구분 뱃지로 메뉴 표시
- 옵션 다중 선택 (샷 추가, 연하게 등) 및 가격 자동 계산
- 메뉴에 없는 경우 기타 직접 입력 가능
- 주문 확인 팝업 후 제출

### 주문 목록
- 오늘의 주문 실시간 조회
- 이미지로 저장 기능 (카페 제출용, 이름/반 제외)

### 오늘의 픽업
- 매일 11:40 주문 마감과 동시에 자동 추첨
- 주문자 중 1명을 랜덤으로 선정해 픽업 당첨자 표시
- 자세히 보기로 전체 추첨 결과 확인 가능
- Firebase 보안 룰로 재추첨 방지

### 기타
- 매일 11:40 자동 마감, 마감 후 주문 폼 비활성화
- 날짜별 주문 데이터 Firebase에 영구 보존

## 기술 스택

| 구분 | 기술 |
|---|---|
| Frontend | React + TypeScript (Vite) |
| Database | Firebase Firestore |
| 배포 | GitHub Pages |

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포

```bash
npm run deploy
```

## 프로젝트 구조

```
src/
├── components/       # 공통 컴포넌트 (폼, 모달, 뱃지 등)
├── constants/        # 옵션 상수
├── context/          # 주문 전역 상태 (OrderContext)
├── data/             # 메뉴 데이터 (menuData.json), 패치노트
├── pages/            # 메인 페이지, 주문 목록 페이지
└── services/         # Firebase 연동 (주문, 추첨)
```
