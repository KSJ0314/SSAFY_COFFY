# ☕ SSAFY COFFEE

> SSAFY 광주캠퍼스 메가커피 단체 주문 관리 서비스

## 소개

매일 점심 전 팀원들의 메가커피 주문을 한 곳에서 모아 관리하는 웹 서비스입니다.<br/>
키오스크 스타일의 메뉴 선택, 실시간 주문 목록 조회, 오늘의 픽업 당첨자 추첨, 문의 게시판까지 지원합니다.

## 주요 기능

### 주문하기
- 이름, 반 입력 후 메뉴를 키오스크 버튼으로 선택 (재방문 시 자동 기입)
- 카테고리 탭으로 메뉴 분류, 선택 시 이미지 표시
- ICE / HOT 통합 버튼 — 온도 전환 시 이미지·가격·칼로리 실시간 변경
- 메뉴 검색 (자동완성, 방향키 내비게이션)
- 옵션 다중 선택 (샷 추가, 연하게 등) 및 가격 자동 계산
- 장바구니로 여러 메뉴 한 번에 주문
- 메뉴에 없는 경우 기타 직접 입력 가능

### 주문 목록
- 오늘의 주문 실시간 조회
- 이름순 그룹화, 복수 주문 시 토글로 펼쳐보기
- 이미지로 저장 기능 (카페 제출용, 이름/반 제외)

### 문의 게시판
- 문의 작성 / 수정 / 삭제 (비밀번호 인증)
- 댓글 작성 / 삭제
- 관리자 답변이 달린 문의는 삭제 불가
- 페이지네이션 (10건 단위)

### 오늘의 픽업
- 매일 11:40 주문 마감과 동시에 자동 추첨
- 주문자 중 1명을 랜덤으로 선정해 픽업 당첨자 표시
- 자세히 보기로 전체 추첨 결과 확인 가능
- MatterMost 웹훅으로 당첨자 알림 자동 전송 및 멘션
- Firebase 보안 룰로 재추첨 방지

### 데스크탑 앱 (Windows)
- 웹 브라우저 없이 Windows 트레이에서 바로 사용
- 트레이 우클릭으로 주문/장바구니/주문 목록/문의/픽업 추첨 각 창 바로 열기
- 이름·반·비밀번호 로컬 저장 (재입력 불필요)
- 픽업 당첨 시 Windows 알림 자동 표시
- 컴퓨터 시작 시 자동 실행 설정
- 새 버전 자동 감지 및 다운로드 안내

### 기타
- 매일 11:40 자동 마감, 마감 후 주문 폼 비활성화
- 날짜별 주문 데이터 Firebase에 영구 보존
- 패치노트 모달 (신규 패치 미확인 시 알림 뱃지 표시)
